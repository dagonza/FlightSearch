### FlightSearch

Usage:
- java -jar flightSearch.jar 
    "number of adults"
    "number of childs"
    "number of infants"
    "id airport origin"
    "id airport destination"
    "date of departure (dd-MM-yyyy)"

Assumptions made :

- I use log4j to print console output.
- I use mockito, hamcrest and junit for testing. All of them library for testing.
